Para funcionar, faça as seguintes instruções:

1- Entre no arquivo .env;

2- Substitua os campos:
	
	meuemail@exemplo.com -> SEU EMAIL DE LOGIN
	minhasenha123 -> SUA SENHA DE LOGIN

É necessário instalar o Node.js (link para instalação: http://nodejs.org/dist/v24.11.0/node-v24.11.0-x64.msi)
